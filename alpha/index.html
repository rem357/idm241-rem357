Microinteraction: Ringer Toggle in Control Center

Riley Mitchell • IDM241

INTERACTION

Platform
Mobile - iOS (iPhone)
Feature
Turning the ringer on/off in Control Center
Description
The ringer can be turned on or off from the Control Center by swiping down from the top-right corner of the screen. In the Control Center, tap the bell icon to control Silent Mode. When the bell has a line through it, Silent Mode is on and the device will not ring for calls or alerts. When the bell is not crossed out, the ringer is on and sounds will play for notifications and incoming calls.

TRIGGERS

  • The user swipes down from the top right of the screen to open the Control Center
  • The user taps on the Ringer/Silent bell icon that indicates the current state

RULES

  1. If state = Ring, tap → Silent; if state = Silent, tap → Ring.
  2. Focus/Do Not Disturb can still mute notifications even if ringer is on.
  3. Hardware switch or Action Button can override Control Center state.
  4. State persists until changed again.

FEEDBACK

Visual

  • Icon toggles between bell and bell-slash.
  • Color and label reflect mode.
  • Banner confirmation that indicates on/off state.

Auditory & Haptic

  • Short tone when enabling Ring (if the volume is on).
  • Haptic tap confirming toggle.

LOOPS & MODES

  • Loop: Toggle is repeatable; the last state is remembered.
  • Modes: Focus or hardware switch create layered states; can conflict with Control Center display.
  • Error prevention: Shows clear message like “Silent due to Focus: Do Not Disturb.”
