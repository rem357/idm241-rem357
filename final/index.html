CONTROL CENTER BUTTON

INTERACTION

Expand the Control Center to reveal a vertical stack of four buttons

TRIGGERS

  • Control Center group button
    • First tap — Collapsed → Expanded
    • Second tap — Expanded → Collapsed
    • Hover ON/OFF button

RULES

  • Hover ON
    • Cursor changes to finger
  • Hover OFF
    • Finger changes to cursor
  • First click: Collapsed → Expanded
    • Control button expands horizontally, and then grows vertically
    • Four child buttons are revealed as button expands
    • Each child button keeps its own independent state from the previous interaction
  • Second click: Expanded → Collapsed
    • Control button shrinks vertically back to compact circle/button
    • Child buttons get covered up when the control button collapses

FEEDBACK

  • Hover ON
    • Cursor changes to finger
  • Hover OFF
    • Finger changes to cursor

    • Control button expands horizontally, and then grows vertically

SCREEN ORIENTATION LOCK

INTERACTION

Locking/unlocking screen orientation.

TRIGGERS

  • Click Orientation Lock button
    • First click — Orientation UNLOCKED → LOCKED.
    • Second click — Orientation LOCKED → UNLOCKED.

RULES

  • Hover ON
    • Cursor changes to finger
  • Hover OFF
    • Finger changes to cursor
  • First click — Orientation UNLOCKED → LOCKED
    • Set state to UNLOCKED (device rotation is allowed)
    • Button background fades to white
    • Phone and lock icon switch to red/light red
    • Icon slightly tilts to rotated phone and then bounces back with the lock icon closed
  • Second click — Orientation LOCKED → UNLOCKED
    • Phone and lock icon rotate in button
    • lock icon becomes unlocked
    • background fades to green-grey, and icons fade to white
    • Icon straightens to upright phone with closed lock

FEEDBACK

  • Hover ON
    • Cursor changes to finger
  • Hover OFF

  • Second click — Orientation LOCKED → UNLOCKED

LOW POWER MODE

INTERACTION

Turning Low Power Mode on/off

TRIGGERS

  • Low Power button
    • First click — Low Power OFF → Low Power ON
    • Second click — Low Power ON → Low Power OFF
    • Hover ON/OFF button

RULES

  • Hover ON
    • Cursor changes to finger
  • Hover OFF
    • Finger changes to cursor
  • First click: Low Power OFF → Low Power ON
    • The outer circle changes from dark grey/green to white
    • The battery outline and tip change to yellow
    • The inner charge bar expands to a longer yellow bar and then shrinks down slightly
  • Second click: Low Power ON → Low Power OFF
    • The outer circle changes from white back to dark grey/green
    • The battery outline and tip change back to white/grey
    • The inner charge bar shrinks back to the smaller white bar

FEEDBACK

  • Hover ON
    • Cursor changes to finger
  • Hover OFF
    • Finger changes to cursor
  • First click: Low Power OFF → Low Power ON
    • The outer circle flashes to white
    • The battery turns yellow
    • The inner charge bar expands to a longer yellow bar and then shrinks down slightly
  • Second click: Low Power ON → Low Power OFF
    • The outer circle fades back to dark grey/green
    • The battery returns to the white/grey style
    • The charge bar shrinks back to the shorter bar
